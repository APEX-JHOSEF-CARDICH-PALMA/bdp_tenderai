## 🧠 Overview

TenderAI permite a empresas y consultoras:

- Identificar oportunidades relevantes automáticamente  
- Evaluar el encaje con su perfil  
- Priorizar con scoring basado en datos  
- Detectar gaps para poder competir  
- Tomar decisiones accionables  

> Enfocado en reducir fricción operativa y aumentar probabilidad de éxito.

## ⚙️ Core Flow

```mermaid
flowchart TD
    A[Company Profile] --> B[Matching Engine]
    B --> C[Ranked Opportunities]
    C --> D[Opportunity Detail]
    D --> E[Score & Probability]
    E --> F[Gap Analysis]
    F --> G[Recommendation Engine]
    G --> H[User Action]

🔍 Key Use Case
	1.	Usuario configura perfil de empresa
	2.	TenderAI sugiere oportunidades relevantes
	3.	Se calcula el score de encaje
	4.	Se identifican gaps
	5.	Se recomienda la siguiente acción
