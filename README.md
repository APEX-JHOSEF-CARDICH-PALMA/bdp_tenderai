Repositorio de soporte visual para **TenderAI**, una plataforma SaaS que optimiza el acceso y participación en licitaciones públicas y ayudas.

---

## 🧠 Overview

TenderAI permite a empresas y consultoras:

- Identificar oportunidades relevantes automáticamente  
- Evaluar el encaje con su perfil  
- Priorizar con scoring basado en datos  
- Detectar gaps para poder competir  
- Tomar decisiones accionables  

> Enfocado en reducir fricción operativa y aumentar probabilidad de éxito.

---

## 🎯 Scope del repositorio

Este repo NO contiene backend ni lógica productiva.

Incluye:

- 🧩 Mockups de producto  
- 🔄 Flujos de usuario (UX)  
- 📊 Diagramas funcionales  
- 🎤 Assets para presentaciones
